Pattern Lab — quick guide

Sketch candlestick patterns by hand, then export them or pull in real price data to reshape. Nothing leaves your browser.

Shape candles

  • Drag a candle's body to move it; drag sideways to change its bar
  • Pull the wick dots to set the high & low
  • Pull the body pills to set the open & close
  • Double-click empty space to add a candle, or a candle to flip it
  • Select one to type exact O/H/L/C values

Annotate

  • Level — a horizontal price line
  • Trendline & Arrow — press and drag; endpoints stay draggable
  • Label — click, then type; double-click to edit later
  • Menus add classic Shapes & multi-bar Patterns — click to append or drag onto the chart

Save & share

  • Library saves named patterns in your browser
  • Import real OHLC data — CSV or JSON
  • Export CSV, JSON, or a PNG image
  • The link button copies a URL with the whole pattern encoded in it

Keys

  • V select C candle L level
  • T trendline A arrow X label
  • F fit ⌫ delete Esc deselect
  • ⌘Z undo ⇧⌘Z redo ? this guide
  • Scroll to zoom price · ⌘+scroll zooms bars · alt-drag pans

Pattern Lab is a drawing tool for studying chart shapes — not trading advice.
